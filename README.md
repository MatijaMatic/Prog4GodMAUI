# MAUI Store App

.NET MAUI application using MVVM architecture and FakeStoreAPI.

## Features
- User login
- Product listing
- Category filtering
- Product details
- Shopping cart
- Recently viewed products

## Technologies
- .NET MAUI
- C#
- MVVM
- REST API
- CommunityToolkit.MVVM

## Architecture
- Models
- Views
- ViewModels
- Services
- Converters
